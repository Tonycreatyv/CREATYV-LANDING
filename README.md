# Creatyv.io Landing Page

Welcome to the Creatyv.io landing page project! This project is designed to showcase the automated services offered by Creatyv.io, aimed at helping businesses reduce time and increase productivity.

## Project Structure

The project is organized as follows:

```
creatyv-landing
├── src
│   ├── main.tsx          # Entry point of the application
│   ├── App.tsx           # Main application component
│   ├── index.css         # Global styles and Tailwind CSS directives
│   ├── components         # Reusable components
│   │   ├── Hero.tsx      # Hero section showcasing the main value proposition
│   │   ├── Features.tsx   # Highlights key features of the services
│   │   ├── HowItWorks.tsx # Explains the process behind the services
│   │   ├── Pricing.tsx    # Outlines the pricing structure
│   │   ├── Testimonials.tsx # Displays customer feedback
│   │   ├── FAQ.tsx        # Addresses common questions
│   │   ├── CTA.tsx        # Call to action component
│   │   └── Footer.tsx     # Footer with copyright and links
│   ├── pages              # Page components
│   │   ├── Home.tsx       # Main landing page
│   │   └── DentalConnect.tsx # Subdomain page for DentalConnect service
│   └── assets             # Assets such as fonts
│       └── fonts
├── public
│   └── index.html         # Main HTML file for the application
├── package.json           # npm configuration file
├── tsconfig.json          # TypeScript configuration file
├── vite.config.ts         # Vite build tool configuration
├── tailwind.config.ts     # Tailwind CSS configuration
├── postcss.config.cjs     # PostCSS configuration
└── README.md              # Project documentation
```

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd creatyv-landing
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser and navigate to:**
   ```
   http://localhost:3000
   ```
